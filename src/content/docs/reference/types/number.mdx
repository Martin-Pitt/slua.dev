`number`s are double-precision (64-bit) floating-point numbers.

```slua
-- \[Arithmetic Operators](#arithmetic-operators)
number + number: number
number - number: number
number * number: number
number / number: number
number // number: number -- floor division
number ^ number: number -- exponentiation
number % number: number -- modulus
-number: number

-- \[Assignment Operators](#assignment-operators)
number += number
number -= number
number *= number
number /= number
number //= number
number ^= number
number %= number

-- \[Comparison Operators](#comparison-operators)
number < number: boolean
number <= number: boolean
number > number: boolean
number >= number: boolean
number == number: boolean
number ~= number: boolean

-- \[Conversion](#conversion)
function tonumber(s: string, base: number?): number?
```
