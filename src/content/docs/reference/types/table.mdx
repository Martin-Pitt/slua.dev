
:::note[Tip]
Methods that start with a table argument can also be called using the colon syntax on table literals or table variables:
```slua
local tbl = {1, 2, 3, 200}
local result = tbl:clone():shrink(true):freeze()
-- result is now a frozen, shrunk clone of tbl
```
:::

## Array-like Tables
Tables can be used as arrays:
```slua
local example = {1,2,3,4}
```

Iterating an array:
```slua
for index, value in example do
    print(index, value)
end
```

## Dictionary-like Tables
Tables can be used as dictionaries/maps:
```slua
local example = {key1 = "value1", key2 = "value2"}
```

Iterating a dictionary:
```slua
for key, value in example do
    print(key, value)
end
```
